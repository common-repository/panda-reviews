== Description ==

A review and rating plugin for WordPress. Let your users to post reviews and vote any posts or custom post from your WordPress site.

Features:

* Voting System
* Review System


Admin Features:

* Manage reviews from your dashboard(delete review, mark as read).
* Customize the star icons ( stars, hearts, likes)
* Customize your main color for post's reviews
* Select your posts type( post or any custom post type from your theme) for reviews to show up.



== Installation ==

The quickest method for installing the Panda Reviews is:

1. Visit Plugins -> Upload Plugin -> Choose File "panda-reviews.zip"
2. Click "Install Now"
3. Finally click "Activate Plugin"

If you would prefer to do things manually then follow these instructions:

1. Upload the `panda-reviews` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
